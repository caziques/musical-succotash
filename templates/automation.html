Google Home / IFTTT Setup

1. Go to IFTTT Maker Webhooks, get your key.

2. Create an IFTTT applet: Webhook trigger → Google Home action (e.g. broadcast message).

3. Use the webhook URL: https://maker.ifttt.com/trigger/{event}/with/key/{key}

4. Paste that full URL into the Action field above when creating a rule.
